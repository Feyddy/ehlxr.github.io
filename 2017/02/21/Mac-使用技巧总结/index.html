七、快速打开最小化程序

Command + Tab 切换程序的时候，最小化的程序不自动弹出窗口，解决办法如下：

  • 用 Command + Tab 切换到要打开的程序
  • 先松开 Tab 键，不要松开 Command 键
  • 然后按住 option 键
  • 然后依次松开 Command 和 option 键

如上操作就可以快速打开最小化的程序了。
